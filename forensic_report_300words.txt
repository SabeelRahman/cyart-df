Registry Analysis (RegRipper): I analyzed a sample NTUSER.DAT hive. Using plugins like recentdocs and userassist, I identified simulated artifacts showing file access and program execution during the incident window (10:10-10:12 UTC).

Network Analysis (Wireshark): I captured baseline traffic using Wireshark. I filtered for DNS and HTTP to establish normal behavior patterns.

Malware Analysis (Static & Dynamic): I performed static string extraction on a suspicious executable, identifying C2 URLs and encoded PowerShell commands. Dynamic analysis via Hybrid Analysis of an Emotet sample showed typical behaviors: registry persistence (T1547.001) and C2 communication (T1071.001).
